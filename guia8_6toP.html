¿Qué es un Conjunto?

Es una colección de objetos que comparten una característica especial. Cada objeto es un elemento.

🍎 Conjunto F: Manzanas, Peras, Bananos...

📏 Conjunto Ú: Lápiz, Regla, Borrador...

¡Eres un experto clasificador! 🧐

🎯

Nuestra Misión

  • ✅ Unir e Intersecar conjuntos como un pro.
  • ✅ Encontrar subconjuntos ocultos.
  • ✅ Crear parejas con el Producto Cartesiano.
🥚🥚

Subconjuntos

Es cuando un conjunto pequeño vive dentro de uno más grande.

¡Decimos que B está contenido en A! (B ⊆ A)

🤝

Unión de Conjuntos (A ∪ B)

¡Es invitar a todos a la misma fiesta! Juntamos los elementos de ambos grupos.

🤝 ¡La unión hace la fuerza matemática! 🤝

💎

Intersección (A ∩ B)

¡Solo los que tienen ALGO en común! Es la parte donde los círculos se cruzan.

¿Qué elementos están en ambos? ¡Esa es la intersección!

👫

Producto Cartesiano (A × B)

Haz clic en los elementos de A para ver sus parejas:

Conjunto A: { 1 }
Conjunto A: { 2 }
Conjunto B: { x }
Conjunto B: { y }

🌈 ¡Cada flecha crea una nueva pareja! 🌈

🌳

¿Cómo ver las parejas?

Usamos un diagrama de árbol para no olvidar a nadie:

🔵 Elemento 1 de A ➡️ {se une con b1, b2, b3...}

🔴 Elemento 2 de A ➡️ {se une con b1, b2, b3...}

🔍
